MSC¶

Plot a number of function with filled gaps between.

This code make creating of multifunctional plot easier to depict. It filling the gaps between multiple functions using a colormaps defined in matplotlib

Created on 01 June 2018 @author: Sergei Shumilin

first_graph_for_article.add_inscription()¶

Add inscriptions to graph: title, labels for x and y axes.

first_graph_for_article.fill_areas()¶

Fill areas between graphs with different colors.

Colors are selected by defining colormap from matplotlib set

first_graph_for_article.plot_a_function(q, k, color)¶

Plot the number of asymptotes and functions according to the q array.

Plots the k-th function thicker then others.

Parameters:
  • q – array of values for x axis
  • k – the exact number of function which need to be plotted thicker then others
  • color – desired color of functions
first_graph_for_article.plot_text()¶

Plot the inscription for the graphs on the plot.

first_graph_for_article.set_tick_labels(xax, number_of_labels, size)¶

Set ticklabels of the given axis.

For rendering of x labels the TEX syntax is used.

Parameters:
  • xax – The axes for which the labels are set
  • number_of_labels – the desired number of labels to put into labels array
  • size – desired size of labels
first_graph_for_article.set_x_values(xax, locator, formatter_pattern)¶

Set the values for the x axis.

Parameters:
  • xax – the desired axis
  • locator – locator sets the multiplicity of x values
  • formatter_pattern – the form of x values
